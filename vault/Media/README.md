# Media Index

Tracked media files and URLs from the Epstein investigation.

**Supported formats (Obsidian native):**
- Images: `.png` `.jpg` `.jpeg` `.gif` `.webp` `.svg` `.bmp`
- Video: `.mp4` `.webm` `.ogv` `.mov` `.mkv` `.avi`
- Audio: `.mp3` `.wav` `.ogg` `.flac` `.m4a` `.aac`
- Documents: `.pdf`
- AI/generated: `.webp` `.avif`

**How to embed in notes:**
```
![[Media/filename.mp4]]           ← local file in vault
![description](https://url.mp4)  ← external URL
```

**DOJ Media Portal:** https://www.justice.gov/epstein
- 180,000+ images released
- 2,000+ videos released
- Access: public, no login required
