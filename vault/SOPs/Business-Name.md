# SOP — Business Name

## Checklist
- [ ] Search Companies House (UK) → https://find-and-update.company-information.service.gov.uk
- [ ] Search SEC EDGAR (US) → https://efts.sec.gov/LATEST/search-index
- [ ] Search OpenCorporates → https://opencorporates.com
- [ ] Google the business name → [[SOPs/Search-Engine]]
- [ ] Check LinkedIn company page
- [ ] Look up domain → [[SOPs/Domain-Name]]
- [ ] Check registered agent / directors
- [ ] Check business address → [[SOPs/Physical-Address]]
