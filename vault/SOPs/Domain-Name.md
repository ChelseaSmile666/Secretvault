# SOP — Domain Name

## Checklist
- [ ] Archive/historical lookup → https://web.archive.org
- [ ] DNS lookup → [[SOPs/DNS-Search]]
- [ ] Check infrastructure → [[SOPs/Computer-Infrastructure]]
- [ ] Run in search engines → [[SOPs/Search-Engine]]
- [ ] WHOIS current → https://whoxy.com
- [ ] WHOIS history → https://whoishistory.com
- [ ] Certificate transparency → https://crt.sh
- [ ] Subdomain enumeration → https://dnsdumpster.com
- [ ] Reverse WHOIS (same owner) → https://viewdns.info

| Tool | URL | Purpose |
