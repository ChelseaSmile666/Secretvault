# SOP — Image Analysis

## Checklist
- [ ] Extract metadata (EXIF) → https://exifmeta.com or `exiftool`
- [ ] Reverse image search — Google Lens
- [ ] Reverse image search — Yandex → https://yandex.com/images
- [ ] Reverse image search — TinEye → https://tineye.com
- [ ] Geolocation from image context → [[SOPs/GPS-Coordinates]]
- [ ] Check for steganography (if suspicious)
- [ ] Run through [[SOPs/File]] checks
